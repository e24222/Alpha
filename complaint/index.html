首次設定 — 連接雲端資料庫
需要 Supabase 帳號才能讓多人共用資料（免費）
1
前往 supabase.com 免費註冊，建立新專案（Free tier）
2
在 Supabase 左側選 SQL Editor，貼上以下 SQL 並執行：
-- 建立客訴資料表 CREATE TABLE IF NOT EXISTS complaints ( id text PRIMARY KEY, date text, customer text, contact text, phone text, email text, product text, lot text, ship_date text, qty integer, order_no text, po text, category text, severity text, description text, demand text, handler text, status text DEFAULT '新案件', note text, logs jsonb DEFAULT '[]', media_count integer DEFAULT 0, created_at timestamptz DEFAULT now(), updated_at timestamptz DEFAULT now() ); ALTER TABLE complaints ENABLE ROW LEVEL SECURITY; CREATE POLICY "allow_all" ON complaints FOR ALL TO anon USING (true) WITH CHECK (true);
3
到 Storage → New bucket，建立名稱 complaint-media，勾選 Public
4
到 Settings → API，複製 Project URL 和 anon/public key 填入下方
Supabase Project URL
Supabase Anon Key
無法連接，請確認 URL 和 Key 是否正確
